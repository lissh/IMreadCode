<span style="">“武冲，让他们聊聊！”身后的陆强突然开口制止了武冲的粗口。武冲蛮横地瞪了绍辉一眼，把身子侧到一旁。</span>
<span style="">绍辉抬头看了看阴云渐起的黑夜，又看看吴晓筱这张熟悉而又姣好的面容，轻轻将她手拉起，吴晓筱没有反抗，顺从地跟着他来到一旁。</span>
<span style="">绍辉心中又是一酸：“晓筱，我没有脚踏两只船，那双鞋是雨嘉的，雨嘉出现在我面前，或许事情发生得太巧你不会相信，但我可以发毒誓来证明……对不起，我不知道自己对你的感情到最后成为伤害你的工具。如果有可能，这辈子我会用命来补偿你，如果没机会，下辈子……”</span>
<span style="">“下辈子，太远了……”吴晓筱悠悠地说道，“这辈子你爱过我吗？”</span>
<span style="">“爱过，我想是的……不过，对你和对雨嘉……我……之前有很多次我对自己下狠心，不如放弃寻找雨嘉和你厮守一辈子……只是我这辈子首先遇到的是她……让我</span><span style="">忘记她……我办不到……我这辈子做的最疯狂的两件事……便是上次瞒着你去四川找她，第二件便是爱上你……”绍辉拉着吴晓筱的手，字字发自肺腑。</span>
<span style="">吴晓筱叹了口气：“也罢，一段感情想修成正果要靠前几世的修行。还好，虽是有缘无分，最起码咱们的生命线有了交集，我知足了。你是对的，分散这么多年还能对女朋友不离不弃，这一点，我没看错你。”</span>
<span style="">“你要真能这么想，那太好了。”绍辉低头说道。</span>
<span style="">“绍辉，我能问你一句题外话吗？”</span>
<span style="">“你说，问什么都行。”</span>
<span style="">“我爸爸曾经念叨了很久，说曾经有一批特种兵去国外执行任务，不知道他们都经历过什么，只知道他们的结局很惨烈，到最后只回来四个，其中一个还残疾了。他四处打听过这几个人，想要他们来这儿工作，你能帮我爸找到他们吗？算是还他老人家一个心愿。”吴晓筱盯着绍辉的眼睛，说出了这个含义很复杂的问题。</span>
<span style="">“他们经历了很多次战争的洗礼，但现在都已经不在了。”绍辉的回答很简洁，也很模棱两可。</span>
<span style="">“你刚才已经答应我这辈子要弥补对我的伤害，现在我正要有事求你，你不会当场拒绝吧？”</span>
<span style="">“你说，就算我办不到我也会去办！”</span>
<span style="">“这几天我爸爸就要对小刀集团动手，但是找不到能够担当要害位置的人选，你能帮助他吗？”</span>
<span style="">绍辉犹豫了，他已经</span><span style="">答应雨嘉明早进藏还愿，看着雨嘉幸福甜蜜的表情，他不忍心爽约。况且，这种动枪杀人见血的事情，他真的已经疲惫了。不过只是一瞬间绍辉便有了最佳人选，他点点头应允道：“好，如果遇到难题，抓紧时间给我打电话，我保证人员第一时间赶到。”</span>
<span style="">吴晓筱细心地看着绍辉的一切，每一根发梢、每一根眉毛，从头至尾看得仔仔细细。因为她突然有了一种预感，今晚分离后，这将是与他最后一次见面。“好吧，但愿那天能赶过来的是你，而不是左明。”</span>
<span style="">其实绍辉心里的最佳人选是赵正豪，因为他是野战特种部队狙击手出身。通俗点讲，在狙击方面，赵正豪是硕博连读毕业，左明是本科，而自己只是专科，且是属于那种经历了坎坷跌宕的挂科生涯后方才毕业的学生。</span>
<span style="">“我不建议现在动手，”绍辉想了想，说道，“看这天气应该会下雪，一旦下雪将会影响狙击手的视线和精度，并且不容易隐藏突击队员的行踪。如果还有时间的话，我建议下雪时安排暗哨和狙击手到位，等雪停之时立刻清扫积雪，让突击队员化装成清洁工，这样可以大大方方毫无痕迹地靠近目标建筑然后发起攻击。还有一点……”他顿了顿，“小刀的瀚海大厦构造复杂而且造型奇特，如果事前不找个类似的模拟建筑演练几次就贸然发动攻击的话，我怕后果</span><span style="">会得不偿失。”</span>
<span style="">“你的建议我会转告给我爸爸，还是那句话，我希望到时出现的会是你。”寒夜里伤心的吴晓筱面对绍辉，眼神依然透露出点点柔情。</span>
<span style="">“吴局现在根本不会听我的意见，否则，”绍辉看了一眼那边的陆强和武冲，“他也不会派他们来我这儿找你。”</span>
<span style="">“好吧，祝你和雨嘉一路顺风，不要因为我而影响你的心情。”吴晓筱知道再说下去终归无用，不如大大方方说好分手词，将伤害和痛苦降到最低。</span>
<span style="">“谢谢，你也一样，好男人很多，你会很幸福的，真的！”绍辉诚恳地说道。</span>
<span style="">吴晓筱莞尔一笑，抬手抚摸了下绍辉的脸庞，转身离开。绍辉恋恋不舍地看着她的背影，却看不到泪水已经挂满她的脸颊。</span>
<span style="">武冲恶狠狠地指指绍辉，跟随着吴晓筱离开。</span>
<span style="">没了吴晓筱的街道，绍辉突然觉得很冷。</span>
<span style="">回去后，绍辉躺在床上给赵正豪打电话，赵正豪说近期不会在云城，问他有什么事，绍辉说只要你没什么事，我们战友就没什么事了。没等赵正豪再说话，绍辉直接挂线然后又给左明说了这事。</span>
<span style="">醉酒后的左明满口应许：只要大哥一声喊，小弟操刀上去砍。绍辉强调，是城市近距精度狙，左明说知道了，距离再近一些他拿子弹扔也能扔十环。绍辉决定明天再给他说这事。</span>
<span style="">次日，雨嘉在火车上一直追问绍辉嘴上的伤是怎么回事。绍辉说昨晚</span><span style="">太冷冻的，雨嘉说不像是冻的，倒像是被人打的，绍辉说你觉得我这身功夫谁能把我打成这样？雨嘉觉得也是。过了不到两分钟，雨嘉又忍不住说看着还是像人用拳头打的，绍辉感觉再这样继续下去，这个问题能够伴随着他去爬唐古拉山口。于是，他决定转移话题。</span>
<span style="">“雨嘉，告诉你件事情，这次去还愿我不是请假来的，我直接辞职了。”</span>
<span style="">“啊！”雨嘉瞪大眼睛不相信地看着他，“你真的辞职了？”</span>
<span style="">“是，我去做警察也只为方便找你，现在找到你了，你又说因为这工作咱俩有了差距，所以，我干脆不干了！”绍辉说到这儿，兴奋地转过身握住雨嘉的手，“雨嘉，我现在也是没有工作的人了，现在你觉得咱俩般配了吗？”</span>
<span style="">“你……真的就为这原因辞职了？”雨嘉一脸感动。</span>
<span style="">“真的，只要能跟你在一起，到哪儿都是家。你放心，我绍辉无论干什么都能养得起你，我还要把你以前所失去的幸福全部还给你！”绍辉一脸坚定。</span>
<span style="">“辉……你太好了……”雨嘉已是泪珠盈盈，“我真的……太幸福了……”她情不自禁投入绍辉的怀中。</span>
<span style="">在他们离开云城的这一刻，这座沿海之城已经飘飘扬扬开始落雪。在这片冰清玉洁的雪空下，吴哲雄对着满窗晶莹深思很久。吴晓筱是自己的女儿，她有多大本事多少见识自己这个当爹的一清二楚。说实话</span><span style="">，她能依据近期天气提出这样独到犀利的战术意见，第一是有高人指点；第二是女儿这贴身小棉袄的的确确心疼爹，想自己之所想，不知想了多久才能想出这种非常专业的意见。</span>
<span style="">但是，吴哲雄最终还是没有采用吴晓筱的建议，原因很简单：第一，这么多年了，他已迫不及待地想把罪魁祸首绳之以法；第二，手下干将和武警支队经过几番演练后，个个拍着胸脯请战，说在自家地盘的建筑物里逮一个人和瓮中捉鳖没什么区别。</span>
<span style="">看着各警种充满激情的战斗力，吴哲雄秘密下达了凌晨两点发起攻击的作战命令。</span>
<span style="">此时，绍辉和雨嘉所在的那列正呼啸着挺进西藏的列车经过了数站的走走停停，车内旅客变化了不少，直到越来越少，毕竟冬季选择进藏的人不是很多。绍辉在车内给彭政委打了一个电话，告诉他自己辞职的事情，彭政委豪爽地说：“自古英雄难过美人关，为红颜而辞掉工作不算什么事，尽管放手去过想过的日子！记住，有困难，找部队，只要部队能帮忙，绝无二话！”</span>
<span style="">话音刚落，绍辉就把云城可能会发生一场大战的事情告诉了他，彭政委问左明是不是还在那里，绍辉给了肯定答案后，彭政委表示会通过吴局和左明密切关注动向，让他放心去雪域遂愿。</span>
<span style="">放下电话，绍辉拥着雨嘉，耳朵和眼睛却担心着放在身边的手机，怕它</span><span style="">随时会响起，给自己带来不希望听到的消息。</span>
<span style="">午夜两点来临，当火车上鼾声四起时，远在另一方的云城却变成了枪火的不夜城。</span>
<span style="">这是警匪之间一场火力相差悬殊的战斗。</span>
<span style="">在这之前，警方制定了缜密的作战方案，每个队员、每个位置甚至每一个动作都进行过细致推敲和演练。其间，由于瀚海大厦的结构和面积过于庞大，关于中心点狙击手的人选吴哲雄曾问过那些特战队员，十五个小伙纷纷表示狙击这种精细活自己并不擅长，如果没有其他素质过硬的人来担任，自己可以扛枪顶上，但是管射不管杀。</span>
<span style="">吴哲雄无奈，只好把他们全部编入首批突击队进行强攻，而所有的狙击手全部由现役的武警队员来担任。准备了这么久，看着这一批批飒爽的武装小伙和神情冷峻的表情，还有自己面前这几套主战方案和备用方案，吴哲雄在大战来临前多少把心放下了点，这些力量，打个小型武装集团应该绰绰有余。</span>
<span style="">但是，他没有想到小刀集团的警备心和火力是如此之猛。悍匪们竟然在周边居民区购有临街房，以普通百姓的身份居住在内作为外围暗哨。进攻队员们借着雪夜秘密抵近时已经暴露了行踪，只好改变方案集中力量进行闪袭强攻。</span>
<span style="">小刀在遁人大厦之前便做好了长期避难的准备，凭借着边境方面给予的准军事配置的强悍火力，竟然在大厦内</span><span style="">部与这些受过专业训练的突击队员打了个平手。当然这也要归功于司空搏的训练与主场作战的优势。</span>
<span style="">瀚海大厦自建成后对外始终处于封闭状态，警方对于内部结构的了解也仅限于图纸。这样一来，几十名突击队员进入大厦发生了短时间火并后方才发觉，自己这是在和不知数量位置的影子们开了仗。一时间大厦内走廊与楼梯处枪声不绝硝烟弥漫，伴随着墙皮的脱落和玻璃的击碎，各种呼叫和支援在黑暗中此起彼伏，脚步急促中伴随有踉跄和人体倒地的沉闷声响。</span>
<span style="">前指指挥官手里捏了一把汗，电台刚刚传来请求支援的呼叫时，指挥官立刻把第二梯队派了进去。第二梯队队员的进入好像是往火堆里丢了把干柴，传过来的枪声更加密集了。</span>
<span style="">小刀没有把这些警察放在眼里，这座大厦在建造之初就是参照古代一些阵法和城池的防御理念而设计的，自己的卫队日复一日在里面演习了这么久，有足够的能力抵挡所有的进攻。当然，对方要是偷懒直接上火炮的话，那就另当别论了。但是这样一来，居住在这座大厦里面的员工可就被殃及生命，他们可是小刀以包吃住的名义精心安排在身边的人质，关键时刻可以救自己一命。</span>
<span style="">听着下面密密麻麻的枪声，他干脆舒服地把腿搭在宽绰的桌面上，抽着烟把玩着手中那把以色列制乌兹冲锋枪，时不时</span><span style="">空瞄几次，嘴里发出“啪啪”声响。</span>
<span style="">时间过去一个小时了，本来计划半小时结束战斗的吴哲雄此刻手心冒了汗。两梯队整整八十个人，强攻一座大厦竟然不得反而困在里面无法动弹。前指反馈回来的消息是：主攻力量已出现伤亡现在转攻为守，请指示。</span>
<span style="">吴哲雄走出指挥车看着漫天雪花，心里也似这天气般阴沉压抑。首攻不善前景不明，再继续下去伤亡可能剧增，撤出来再重新组织进攻的话，无疑是将己方的时机优势和战斗信心丢失殆尽。</span>
<span style="">是增派力量继续强攻还是命令撤出？吴哲雄正在心里权衡着。</span>
<span style="">武警方面的参谋组给出的意见是继续进攻，一鼓作气拿下这场战斗。</span>
<span style="">以陆强和刑警支队长为首的警方认为已经打草惊蛇，不如先撤出这泥潭等天亮再商议，重组兵力或者向上级汇报申请派特种部队前来参战，这样可以减少不必要的伤亡。</span>
<span style="">吴哲雄最终选择了后者。于是，以那十五个退役特战小伙为首的两个战斗梯队在大厦内开始撤离。等人员撤出正门之时，小刀集团的枪手居高临下进行了倾斜式射击，场面如烟花瀑布般绚丽，身处弹雨之中的队员仓促抬起枪口进行还击。数秒间，眼看全体就要淹没在红色烟火中时，对面的狙击手终于找到了目标发威。枪响数声，子弹精准射进枪手们的身体，随即几具尸体坠落在大厦堂门处。趁着</span><span style="">这个时机，队员们迅速地撤到了安全地带。</span>
<span style="">吴哲雄看望了刚刚撤回的队员，在鬼门关走了这么一圈的他们来到安全地带后身体不由自主地开始抖动。吴哲雄安排了参战队员卸下装备休息后，他站在积雪中叹了口气，知道今晚的作战算是以失败告终了。</span>
<span style="">各警种按照命令加强了包围封锁力量，静静等待着天际发亮和新一轮的战斗来临。</span>
<span style="">雪突然大了起来。</span>
