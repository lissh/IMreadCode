<span style="">赵正豪甩灭火柴，返身坐在沙发上：“我如果有害你的意思，就不会只身过来和你谈生意，就凭这点，你应该相信我。”</span>
<span style="">“既然这样，你也应该知道我和肖华、司空搏之间的关系，你就这么肯定我会跟你合作？”小刀反问道。</span>
<span style="">“虽然你是他们扶持起来的，恕我直言，你只是他们手里可以随意摆布的一个傀儡。毒品的利润巨大，你冒着莫大风险为他们贩毒，得到的却只是残羹剩饭，你心里会平衡吗？现在他们走了，这正是你真正独立的绝佳机会。而且你是一个有自己抱负的人，所以，就凭这点，我相信咱们会合作的。”赵正豪诚恳地说道。</span>
<span style="">小刀沉默了，室内空气凝聚，静止成冰。</span>
<span style="">“他们突然出事，应该不是水鬼暴露的原因吧？”许久，小刀突然问了这个问题。</span>
<span style="">赵正豪沉默不语。</span>
<span style="">“你打算怎么个合作法？”小刀换了个问题。</span>
<span style="">“把咱们的市场合二为一，我负责供货，你负责交易，五五开。”</span>
<span style="">“有现货吗？”</span>
<span style="">“随时有。”</span>
<span style="">“我考虑考虑，肖华那边不是那么好糊弄的。”</span>
<span style="">“好的，不打扰了。”</span>
<span style="">“不送。”</span>
<span style="">道完别，赵正豪轻轻在茶几上放</span><span style="">下一个物件，推门而出。</span>
<span style="">小刀翻转手枪，枪柄内已经空空如也。小刀瞪着茶几，愣了很久。</span>
<span style="">“君浩走了之后，他经常出现在我的梦里，一有时间我就会去他长眠的地方看他。在他墓前，我能听见他对我说的每一句话，他对我的思念，他对我的牵挂……这么多年了，有很多男孩追求我，可是我发现，我最爱的只有他……还有那堆黄土……因为那里面，埋着我最爱的男孩……”</span>
<span style="">吴晓筱仔细打量了绍辉一番，确定他身上没有任何黄土痕迹后，又认真地听那位叫茜茜女孩的诉说。</span>
<span style="">“你们不知道那种感觉，就好像有细沙穿心，我的心隐隐作痛，这种痛连绵不绝，让人看不到希望。于是我知道我这辈子再也离不开他了，即便是他已走了这么多年，所以……”茜茜故作欢颜垂泪一笑，“为了能和他生前的世界贴近一些，我来到了这里。”</span>
<span style="">太突然了，绍辉竟然一时说不出话来。</span>
<span style="">但是吴晓筱却有话了：“这到底是怎么一回事，谁能给我解释一下？”</span>
<span style="">茜茜不认识吴晓筱：“请问您是哪位？”</span>
<span style="">“如果我没说错的话，我是你嫂子！”吴晓筱把“嫂子”俩字咬得很重。</span>
<span style="">“哪个嫂子？”茜茜不明白。</span>
<span style="">吴晓筱伸出胳膊挽住绍辉，手指用力掐着他的胳膊内侧：“绍辉，我是哪一个？”</span>
<span style="">“她想问你是左明的媳妇还是我的媳妇，茜茜，有些字不能省！”绍辉</span><span style="">吃痛，咬牙说道。</span>
<span style="">“不好意思嫂子，我叫聂茜，是明哥和辉哥战友的女朋友，是专门过来投奔他们的。”茜茜笑吟吟地解释道。</span>
<span style="">误会解除了，绍辉的胳膊立刻感到前所未有的舒畅。</span>
<span style="">“左明，茜茜既然来了，你为什么不告诉我？你安的什么心？”绍辉刚刚脱离了生命危险，紧接着拿左明开刀。</span>
<span style="">左明摊开手：“我敢么？我还没挨够教训么？耗子挨了几次打还知道出门前侦查情况呢，给你说，我吃饱了找虐啊！”</span>
<span style="">绍辉心虚地摆摆手，示意他别说了。吴晓筱却又问道：“那个君浩是谁？他怎么了？”</span>
<span style="">“牺牲了。”绍辉淡淡地回答道。</span>
<span style="">“现在是和平时期啊，他是怎么牺牲的？”</span>
<span style="">“演习时枪支走火，打中了他。”一提起刘君浩，心情本来就很沉重的绍辉更加喘不过气。吴晓筱看到绍辉此时的模样，很识时务地闭上了嘴。</span>
<span style="">“茜茜，我跟你辉哥都在这里，以后这就是你的家，我们会好好照顾你的！”左明发自肺腑地说道。</span>
<span style="">“嗯！”茜茜幸福地点点头，“找到了你们，我就感觉君浩重新回到我身边了。”</span>
<span style="">突然间，心情沉重的绍辉感到莫大的释然。记得当年刘君浩在雪山断崖深处临死前曾经对他说过，他最放不下的就是父母，还有茜茜，他曾恳请绍辉把茜茜托付给左明。</span>
<span style="">时过境迁，因为种种因素，绍辉没有完成刘君浩临死前的嘱托，而如今</span><span style="">缘分重来，冥冥中指引着茜茜来到了左明身边。这样一来，绍辉的另一块心病也了结了：茜茜终于有了一个很好的归宿。</span>
<span style="">他决定极力撮合两人在一起，让左明代替刘君浩照顾茜茜一辈子。</span>
<span style="">想到这儿，绍辉心情大好：“摆酒！喝到天亮！”</span>
<span style="">“敢！”吴晓筱杏眼一瞪。</span>
<span style="">“摆酒！天亮再喝！”绍辉输阵不输气。</span>
<span style="">“哎，你们当兵的就是幸福，有那么多漂亮女孩愿意跟随着，你再看看其他男人，费尽千辛万苦才能娶得心上人，大部分还以失败告终，而你，没费一点力气就拥有了我，你知足吗？”回去的路上，柔柔夜风中，吴晓筱略带羞涩地问道。</span>

<span style="">“啊！啊！”旖旎夜色中，响起了绍辉的阵阵哀号。</span>
