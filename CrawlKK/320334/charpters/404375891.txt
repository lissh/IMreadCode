<span style="">这天晚上，绍辉一个人独自坐在海边，看了一夜的潮起潮落。</span>
<span style="">次日，办公室里，被风吹红双眼的绍辉不敢面对哭红双眼的吴晓筱。不知道其中厉害的</span><span style="">王瑞打趣道：“哟，你俩的眼睛还真有夫妻相！”</span>
<span style="">这句话一出口，后果很严重。于是王瑞有自知之明地消失了好几个小时直到下午四点多钟，有人在外面小心地敲敲门，一张颤抖的脸在狭小的门缝里塞了进来：“辉哥，咱们俩出警。”</span>

<span style="">绍辉愣了，世界之大无奇不有。</span>
<span style="">云城市动物园修筑在海落区，与公安局相距不远。片刻，绍辉俩人来到目的地，虽是日薄西山的黄昏，园内依然人声鼎沸。</span>
<span style="">绍辉开始查看现场，这个蛇笼很大，是专门用来放这条蝮蛇的。此时，那条蝮蛇已经转移到其他地方，有一些游客在此围观。当他们向这里的工作人员详细地了解了情况之后，两个人跟着工作人员来到了动物园管理处。等绍辉和王瑞走进去一看，两个人同时傻眼，那个意图谋害蝮蛇的人，竟是左明！</span>

<span style="">经过一番调解，鉴于蝮蛇安全无恙，没有引起太大的乱子，</span><span style="">绍辉又大概解释了左明之所以这么做的原因，于是最后协商赔偿一定数额的钱款，动物园方面决定不再追究。</span>
<span style="">说起刘君浩，当年绍辉和左明，以及其他军种的精英们奉命执行那次机密任务时，曾经在雪山深处发生过一场艰苦卓绝的突围战。对手组织代号“喜马拉雅巨蝮”，队长随身饲养着一条巨型蝮蛇，刘君浩的最后一口气，就是断送在它的嘴里。因此，时隔多年以后，活着回来的左明再次看到这种蛇类时，再也忍不住内心的痛苦……</span>
<span style="">是夜，俩人坐在有些清冷的海边喝了整整一夜的酒。</span>
<span style="">次日，俩人打着哈欠和喷嚏回去睡了整整一天的觉。</span>
<span style="">在生意场上，所有人都需要在觥筹交错中迎来送往增深感情。在把所有精力都投放在毒品市场后，肖华与司空搏仍要挤出部分时间与酒水类的客户推杯换盏，借以向外界证明他们正在努力经营酒水城，来伪装自己的真实身份。其实他们有时候也需要酒精的麻痹，什么样的人看什么样的世界，过着什么样的生活。对于他们而言，在这充满阴冷险毒没有丝毫安全感的世界，女人的身体和浓烈的酒精是一堆烈火，能暂时烘干温暖他们不曾温热过的心。</span>
<span style="">在陪着几个客户和官员灌下八瓶高度白酒之后，所有人都踉踉跄跄地走出高级会所上了各自的车。肖华和司空搏一上车踉跄之态立刻全无，水鬼驾</span><span style="">驶着那辆越野车慢慢滑下厅门，冲酒水城驶去。</span>

<span style="">“那怎么办，甩不掉就这么带着他遛弯？”水鬼不满地说道，“给我十秒钟的时间，我能把他塞进深海喂鲨鱼。”</span>

<span style="">这间单间所在的海宴私人会所驻在云城市最陡峭的海崖边，正所谓无限风光在险峰，在这里真正做到了面朝大海春暖花开。推窗望去，碧波浩海连绵至天涯，这令在海边出生长大的若叶欣喜异常。韩戈更是款款深情注视着面前这个深爱的女孩，丝毫不知道有客正缓缓逼近。</span>
<span style="">陡峭悬崖壁上，司空搏与水鬼正戴着面罩与壁虎手套攀爬。海宴会所作为云城市乃至周边城市最顶</span><span style="">级的私人场所，不只是有钱就可以进出的，但是只要有能力和胆量，会所窗外的峭壁也可以是通往会所内部的一条道路。室内，韩戈正握着若叶的双手深情相视，不料，另一双戴着怪异手套的手出现在观光窗口，随即两个蒙面人一跃而进，吓得若叶花颜尽失。</span>
<span style="">司空搏站稳之后伸出食指示意她不要喊叫，而后站在璀璨水晶灯下仔细打量了室内一番，确定没有摄像头后魔术般掏出一盒包装精美的礼品，毕恭毕敬地呈给若叶。韩戈则波澜不惊地看着这两名不速之客，静等下文。</span>
<span style="">“这是韩老板专门为您安排的一个惊喜，请您笑纳。”司空搏眉目带笑地弯腰说道。</span>

<span style="">若叶优雅地将礼盒打开，一只世界级限量版手表顿时呈现在众人眼前。或许是韩戈送给若叶的礼物都很贵重，若叶只是礼节性地吃惊赞叹一番，这多少令司空搏有些失望。</span>

<span style="">“不急，我也有份礼物送给韩老板。”水鬼在面罩里说道，随后拿出一个盒子递给韩戈。韩戈打开看了看，不动</span><span style="">声色地问道：“礼物很贵重，也很漂亮，但是不符合我的胃口，还是物归原主吧。”说着，他把盒子递了过去。</span>
<span style="">司空搏没有伸手去接：“我和兄弟费这么大周折来此见您一面，您不收下，太不给我们兄弟面子了吧？”韩戈把盒子推给他，缓缓说道：“钱我敢收，枪我也敢收，就算里面是颗人头，我也照收不误，但是这东西我是绝对不碰的！”</span>
<span style="">“兄弟，大家都是出来混的，适当收敛一些，也是为自己留条后路多条活路，也多一条生金之路。</span>
<span style="">你有场面优势，我有货源优势，只要咱们公平合作，对彼此都有好处。”这一次，司空搏不再称呼韩老板，而是冷言冷语地称兄弟。</span>

<span style="">司空搏笑了，韩戈也哈哈笑着拍拍司空搏的肩膀。若叶恍然大悟：刚才他们这么严肃地说话，原来是在谈论工作。</span>
<span style="">“兄弟，以前我们大哥很多次非常有诚意地过去找你商量合作事宜，你一直避而不见，最后干脆交代手下直接把他赶出去。能见你一面真难啊，尤其是还能看到你们小两口在一起，真是一件不可思议的事情。”司空搏特意把“你们小两口”说得很重。</span>
<span style="">聪明的韩戈当然明白这句话的分量，面对着这两台精悍的杀人机器，他脑子正在飞速转动，一只手伸向桌角</span><span style="">的香烟盒准备抽支烟出来。</span>
<span style="">“既然能用这种方式和你见面，你认为这里的保安会是我们的对手吗？”司空搏淡淡地说了一句。</span>
<span style="">韩戈停止了动作，把手在应招按钮处拿了回来，半晌，他慢条斯理地问道：“云城市这么大的一块蛋糕，为什么非要捡我这粒芝麻？”</span>
<span style="">“你错了，”司空搏又笑了，“整个云城市加起来也只不过是一粒芝麻。你才是蛋糕，没有你的加盟，不敢言云城。”</span>
<span style="">韩戈思索良久：“这件事情事关重大，可否容我考虑一段时间，如果你现在就要逼我的话，我的实力你也是知道的，你要的是钱，不是跟我作对。”</span>
<span style="">这句话说到司空搏心里去了，他只求财，根本不想在云城市跟实力雄厚的韩氏家族作对，这样对他没有一丝好处。话已到此，他今天的目的也已经达到，聪明的司空搏知道事情要有个度，他拿出手机：“那劳烦兄弟告诉我你的手机号，两天之后我会跟你联系。”</span>
<span style="">韩戈说出一串号码，司空搏当场打过去，韩戈随身携带的手机响起，司空搏满意地点点头：“这样就好，兄弟，既然这次咱们能聚到一起，如果有什么意外，下次咱们还会用这种方式相见的，你是聪明人懂得我的意思，祝咱们以后合作愉快！”说完，司空搏起身要走。</span>
<span style="">“还是我带你们出去吧，这么高的悬崖，下去要比上来难得多。”韩戈大度地说道</span><span style="">。</span>
<span style="">“你果然仗义，不过你认为我们这一身装扮能安全走过摄像头吗？”司空搏微微一笑，仰身跳出窗口，引得若叶一声惊呼。</span>
<span style="">“再见！”水鬼色迷迷地冲若叶招招手，跟随着司空搏一跃而下。悬崖间，两张降落伞盛开在半空，飘摇而下。</span>
<span style="">韩戈对着若叶甜美一笑，悄悄擦了擦掌心里的汗水。</span>
<span style="">海边，司空搏两人收起降落伞后拿出手机，把韩戈的电话号码记下。司空搏抡起手臂高高一扬，手机呈抛物线落入海中，“砰！”他张开双臂夸张地模仿爆炸声。</span>
<span style="">“哥，你觉得有戏吗？”水鬼也被他的情绪感染。</span>
<span style="">“兄弟记住，只要有演员，就会有戏！”司空搏大笑着拍拍水鬼，冲自己的停车处走去。</span>

<span style="">四名荷枪实弹的干警在山间小道被人赤手空拳地杀害，这种话题无疑是纵贯大街小巷的沙尘暴。虽然警方出于种种考虑把这件事压着不发，但小道消息与各种版本的传说已经是铺天盖地。这已经不是第一次发生警察被害的案件，况且这一次是在月黑风高的山间，更为这次特大案件蒙上一层神秘色彩。</span>
<span style="">有人说是野兽所为，有人说是山神所为，有人说是一伙人围殴四个警察，警察因寡不敌众而被打死。更有人信誓旦旦地说自己曾亲眼看到那山里面有古代山贼的亡灵，是它们杀害了那些警察，否则四个受过训练的持枪警察，怎么会吭都不吭地连一丝反抗都没有就死掉？众说纷纭，一时间恐怖气氛笼罩在每个云城市民的头上。</span>
<span style="">吴哲雄黑着脸连续参加了四场葬礼，这位军人出身的老局长隐约感觉到在这片已不再熟悉的土地中，隐藏着一些身怀绝技而又仇视社会的亡命徒，如果不把这几个人找出来，类似的案件还会继续出现。但是，通过这几个案件来看，他们的手段异常老练，像是这片海域的鱼，偶尔在深海里翻出一朵浪花后又从容地消失在海水里。</span>
<span style="">该怎么捕鱼呢？吴哲雄站在送行的队伍中苦苦思索着。</span>
<span style="">市井中的各种传说绍辉都听说了，陆强在讲案发现场和被害警察伤口时他也仔细听了听。很显然，这绝不是什么鬼神作怪，也不是</span><span style="">多人作案，倘若人多了，被害人的伤口绝不会这么少而精准。凶手最多俩人，若有高手，一人也能办到。单身瞬间击毙四名武装人员，不只是功夫好就能做得到，他和他的很多战友都有这能力，一些受过特别训练的，顶级保镖也可以。总之，能做出这种恐怖案件的人绝对拥有特殊背景，可以着手在这里寻找破案口。</span>
<span style="">绍辉皱着眉头思索着，他认为有必要把这些想法向陆强汇报一下。正准备上楼找陆强时，一只纤纤玉手搭在他肩膀，绍辉扭头一看，神经立刻高度紧张起来，低头哈腰道：“科长您早！”吴晓筱手搭起凉棚眯眼看看太阳，绍辉这才意识到自己问候的时间不对：“科长下午好！”</span>
<span style="">“嚯，一天就这么快飞逝了，那老娘的大姨妈岂不是刚送走紧接着又要来了？”自从绍辉数次招惹到她之后，吴晓筱的嘴里全是噎人的火药味。</span>
<span style="">“科长您找我什么事？”绍辉见来者不善，立刻转移话题，“要没什么事的话，我先去忙了。”说着就想拔腿跑路。</span>
<span style="">“站住！”吴晓筱眼睛一瞪，“全世界的警察都在忙着破案抓凶手，就你游手好闲站在院子里望天，你不觉得愧疚么？”</span>
<span style="">“那我该怎么办？”绍辉听话地站住，反问道。</span>
<span style="">“我跟我爸爸说了，准备把你调进市刑警队，你不用感谢我。”吴晓筱说完，转身就走。</span>
<span style="">绍辉大急，他的心思本来就</span><span style="">没放在工作上，混进警察队伍的目的就是想当个小民警，借工作之便来寻找自己的那个雨嘉。倘若把他调进天天跟大案凶案打交道的刑警队，那刑警队长跟他之间，俩人必有一疯。再说，他已厌倦了暴力相向的岁月，自己一旦再跌进去，那绝对是度日如年。所以，他急忙追上去，痛哭流涕地表示自己是三代单传，也没那本事面对穷凶极恶的歹徒，进刑警队等于是绝了绍家的后。</span>
<span style="">而吴晓筱却一脸恨疯了的表情看着绍辉：“这是我赔着笑给你争取来的机会，你看看现在的形势，市刑警队是很容易立功的地方！你不去，你良心被狗吃了？老娘我这么矜持的一个黄花大姑娘牺牲了色相为你换来这么一个机会，你就这么欺负老娘？你以为老娘是李香君么？”</span>
<span style="">“去还是不去？”</span>
<span style="">绍辉果断地摇头。</span>
<span style="">“就算为我，去不去？”</span>
<span style="">绍辉又摇摇头。</span>
<span style="">“绍辉，别仗着我喜欢你，你就可以为所欲为屡次伤害我，你以为你是谁？”吴晓筱咬着嘴唇，泪眼婆娑的有些激动。</span>
<span style="">绍辉沉默不语。</span>
<span style="">“绍辉，你不是个男人！我吴晓筱看错你了！”吴晓筱伤心地喊了一句，头也不回地跑回办公室，“砰”一声狠狠摔上门。</span>
<span style="">这天晚上，绍辉一个人独自坐在海边，看了一夜的潮起潮落。</span>
<span style="">次日，办公室里，被风吹红双眼的绍辉不敢面对哭红双眼的吴晓筱。不知道其中厉害的</span><span style="">王瑞打趣道：“哟，你俩的眼睛还真有夫妻相！”</span>
<span style="">这句话一出口，后果很严重。于是王瑞有自知之明地消失了好几个小时直到下午四点多钟，有人在外面小心地敲敲门，一张颤抖的脸在狭小的门缝里塞了进来：“辉哥，咱们俩出警。”</span>
<span style="">和王瑞出警，是绍辉今天最舒服的时刻，不过也非常好奇。绍辉挖苦了王瑞大半路之后才想起问警情。这次的案件很奇怪：云城市动物园刚刚引进来一条长达三米的南美洲巨型蝮蛇，霎时间成为这座海边城市最耀眼的明星，所有人纷纷慕名前来观赏。可是，今天下午却有人意图谋害这条蛇……</span>
<span style="">绍辉愣了，世界之大无奇不有。</span>
<span style="">云城市动物园修筑在海落区，与公安局相距不远。片刻，绍辉俩人来到目的地，虽是日薄西山的黄昏，园内依然人声鼎沸。</span>
<span style="">绍辉开始查看现场，这个蛇笼很大，是专门用来放这条蝮蛇的。此时，那条蝮蛇已经转移到其他地方，有一些游客在此围观。当他们向这里的工作人员详细地了解了情况之后，两个人跟着工作人员来到了动物园管理处。等绍辉和王瑞走进去一看，两个人同时傻眼，那个意图谋害蝮蛇的人，竟是左明！</span>
<span style="">绍辉一下就明白了所有事：左明做这件事一定是为了刘君浩——他们曾经那个生死与共的战友！</span>
<span style="">经过一番调解，鉴于蝮蛇安全无恙，没有引起太大的乱子，</span><span style="">绍辉又大概解释了左明之所以这么做的原因，于是最后协商赔偿一定数额的钱款，动物园方面决定不再追究。</span>
<span style="">说起刘君浩，当年绍辉和左明，以及其他军种的精英们奉命执行那次机密任务时，曾经在雪山深处发生过一场艰苦卓绝的突围战。对手组织代号“喜马拉雅巨蝮”，队长随身饲养着一条巨型蝮蛇，刘君浩的最后一口气，就是断送在它的嘴里。因此，时隔多年以后，活着回来的左明再次看到这种蛇类时，再也忍不住内心的痛苦……</span>
<span style="">是夜，俩人坐在有些清冷的海边喝了整整一夜的酒。</span>
<span style="">次日，俩人打着哈欠和喷嚏回去睡了整整一天的觉。</span>
<span style="">在生意场上，所有人都需要在觥筹交错中迎来送往增深感情。在把所有精力都投放在毒品市场后，肖华与司空搏仍要挤出部分时间与酒水类的客户推杯换盏，借以向外界证明他们正在努力经营酒水城，来伪装自己的真实身份。其实他们有时候也需要酒精的麻痹，什么样的人看什么样的世界，过着什么样的生活。对于他们而言，在这充满阴冷险毒没有丝毫安全感的世界，女人的身体和浓烈的酒精是一堆烈火，能暂时烘干温暖他们不曾温热过的心。</span>
<span style="">在陪着几个客户和官员灌下八瓶高度白酒之后，所有人都踉踉跄跄地走出高级会所上了各自的车。肖华和司空搏一上车踉跄之态立刻全无，水鬼驾</span><span style="">驶着那辆越野车慢慢滑下厅门，冲酒水城驶去。</span>

<span style="">司空搏微闭双眼，片刻后回了一句：“绕圈子。”水鬼猛地一脚油门，越野车改变方向直奔外环，七折八拐地来到人烟稀少的地方后，水鬼又把越野车的速度飙至240迈。令他惊讶的是，后面那辆普桑仍然保持着一定距离紧紧跟随。这时虽看不清细节，但稍微懂车的人也会明白那绝不是一辆普通的桑塔纳。</span>
<span style="">“华哥，这儿没人，做了他吧。”水鬼恶狠狠地说道。</span>
<span style="">肖华缓缓摇摇头：“不行，如果是警方的侦察员，就会呼叫支援，太危险。”</span>
<span style="">“那怎么办，甩不掉就这么带着他遛弯？”水鬼不满地说道，“给我十秒钟的时间，我能把他塞进深海喂鲨鱼。”</span>
<span style="">“等你跟他照面时，警方的支援也就赶来了，拐弯上高速。”肖华说完，闭上双眼。</span>
<span style="">越野车又是一声怒吼，歇斯底里地冲高速路口绝尘而去。</span>
<span style="">车里，肖华跟司空搏稳坐不动，但脑子却和这辆越野车的发动机一样高速运转着。这么多年来他们隐蔽得天衣无缝，从没遇到被人跟踪这种事。退一步讲，对他们而言，被人盯上梢就等于一只脚已经迈进棺材里。</span>
<span style="">今天这不是第一次，自从那晚毒品丢失事件发生后，白夜就首次发现有人在跟踪自己，擅长追踪的他几次想反跟踪，却总是让</span><span style="">对方消失在茫茫人海和交错建筑之间。今天肖华等人又被跟踪，如果是同一个人所为，那说明他们的目标和行踪已被暴露。肖华总是担心因为那批货的丢失而引来警方的注意，如此看来，他的担心恐怕要成为现实了。</span>
<span style="">“华哥，那明天的安排取消吗？”司空搏冷静地问了一句。</span>
<span style="">“依计划进行，明天的机会很难得。”肖华回答道。</span>
<span style="">要说云城市的富二代和纯度最高的钻石王老五，当属韩戈为头牌，标准的个头，霸气中带着一些儒雅，果敢有性格的他也是很多大姑娘小媳妇眼中的白马王子。毫不客气地讲，如果韩戈想放纵自己的话，他可以为所欲为地过夜夜当新郎的生活，闺中有女的人家也希望能攀上这条高枝从而坐地云升。可是，他今天却放弃了众多红颜去机场接自己的女友，在其他人看来，这跟出家为僧没什么区别。</span>
<span style="">这位幸运女孩名叫若叶，来自日本，今天专程也是首次来到这座遥远也贴心的城市来看男友。若叶长相安静甜美，跟电视里看到的那些日本美女很相似。她一颦一笑之间有着樱花落叶般的静美，一举一动如温泉泛着涟漪，令人心醉。韩戈这位感情浪子自从在日本邂逅她之后，就像一个久旅沙漠的迷路者突然看到一片绿洲，连衣服也来不及脱掉一个猛子扎进去，再也不想出来了。</span>
<span style="">若叶喜欢淡粉色，今天韩戈专门为</span><span style="">她穿了一件淡粉色的衬衣，用自己的欧陆跑车换来一名女下属的淡粉色QQ小轿车，吓得那名女下属差点当场摆上香坛把这辆车供起来。</span>
<span style="">韩戈没开过手动挡的车，小轿车一窜一窜地把韩戈送到机场后狂喷一阵烟后戛然熄火。韩戈也轻松不到哪儿去，衬衫被汗水湿透紧贴身体，更显了男性迷人之处。这样一来，韩戈的回头率很高，但是他平时为人比较低调，没有人识得他就是大名鼎鼎的九叔之子。</span>
<span style="">在机场的国际出口，韩戈终于看到了日夜萦绕在自己心头的若叶。久别的恋人再次重逢是一组浪漫的镜头，少不了拥抱亲吻与窃窃私语。当若叶看到韩戈开来的那辆粉红色轿车时，明白了男友的良苦用心，一股甜蜜再次涌上心田，注满了整个车厢，小轿车就这么载满甜蜜又一窜一窜地把这对恋人送回市里。</span>
<span style="">在这里，韩戈早已安排好一间粉红色的私密单间为若叶接风。</span>
<span style="">这间单间所在的海宴私人会所驻在云城市最陡峭的海崖边，正所谓无限风光在险峰，在这里真正做到了面朝大海春暖花开。推窗望去，碧波浩海连绵至天涯，这令在海边出生长大的若叶欣喜异常。韩戈更是款款深情注视着面前这个深爱的女孩，丝毫不知道有客正缓缓逼近。</span>
<span style="">陡峭悬崖壁上，司空搏与水鬼正戴着面罩与壁虎手套攀爬。海宴会所作为云城市乃至周边城市最顶</span><span style="">级的私人场所，不只是有钱就可以进出的，但是只要有能力和胆量，会所窗外的峭壁也可以是通往会所内部的一条道路。室内，韩戈正握着若叶的双手深情相视，不料，另一双戴着怪异手套的手出现在观光窗口，随即两个蒙面人一跃而进，吓得若叶花颜尽失。</span>
<span style="">司空搏站稳之后伸出食指示意她不要喊叫，而后站在璀璨水晶灯下仔细打量了室内一番，确定没有摄像头后魔术般掏出一盒包装精美的礼品，毕恭毕敬地呈给若叶。韩戈则波澜不惊地看着这两名不速之客，静等下文。</span>
<span style="">“这是韩老板专门为您安排的一个惊喜，请您笑纳。”司空搏眉目带笑地弯腰说道。</span>
<span style="">“她是日本女孩，听不懂中国话。”韩戈沉着地回应道，“我来帮你翻译吧。”随后，韩戈用日语把司空搏的话翻译过去，只见若叶的表情由恐惧慢慢转为吃惊继而又高兴，她非常有礼貌地面带微笑起身接过礼盒，道了谢。</span>
<span style="">若叶优雅地将礼盒打开，一只世界级限量版手表顿时呈现在众人眼前。或许是韩戈送给若叶的礼物都很贵重，若叶只是礼节性地吃惊赞叹一番，这多少令司空搏有些失望。</span>
<span style="">“你们的心意我看到了，说正事吧。”韩戈将腿搭起来，端坐沙发椅中。</span>
<span style="">“不急，我也有份礼物送给韩老板。”水鬼在面罩里说道，随后拿出一个盒子递给韩戈。韩戈打开看了看，不动</span><span style="">声色地问道：“礼物很贵重，也很漂亮，但是不符合我的胃口，还是物归原主吧。”说着，他把盒子递了过去。</span>
<span style="">司空搏没有伸手去接：“我和兄弟费这么大周折来此见您一面，您不收下，太不给我们兄弟面子了吧？”韩戈把盒子推给他，缓缓说道：“钱我敢收，枪我也敢收，就算里面是颗人头，我也照收不误，但是这东西我是绝对不碰的！”</span>
<span style="">“兄弟，大家都是出来混的，适当收敛一些，也是为自己留条后路多条活路，也多一条生金之路。</span>
<span style="">你有场面优势，我有货源优势，只要咱们公平合作，对彼此都有好处。”这一次，司空搏不再称呼韩老板，而是冷言冷语地称兄弟。</span>
<span style="">“你们还有事吗？没事就请离开，今天的事情我当没发生过！”韩戈毫不客气地下了逐客令。</span>
<span style="">司空搏笑了，韩戈也哈哈笑着拍拍司空搏的肩膀。若叶恍然大悟：刚才他们这么严肃地说话，原来是在谈论工作。</span>
<span style="">“兄弟，以前我们大哥很多次非常有诚意地过去找你商量合作事宜，你一直避而不见，最后干脆交代手下直接把他赶出去。能见你一面真难啊，尤其是还能看到你们小两口在一起，真是一件不可思议的事情。”司空搏特意把“你们小两口”说得很重。</span>
<span style="">聪明的韩戈当然明白这句话的分量，面对着这两台精悍的杀人机器，他脑子正在飞速转动，一只手伸向桌角</span><span style="">的香烟盒准备抽支烟出来。</span>
<span style="">“既然能用这种方式和你见面，你认为这里的保安会是我们的对手吗？”司空搏淡淡地说了一句。</span>
<span style="">韩戈停止了动作，把手在应招按钮处拿了回来，半晌，他慢条斯理地问道：“云城市这么大的一块蛋糕，为什么非要捡我这粒芝麻？”</span>
<span style="">“你错了，”司空搏又笑了，“整个云城市加起来也只不过是一粒芝麻。你才是蛋糕，没有你的加盟，不敢言云城。”</span>
<span style="">韩戈思索良久：“这件事情事关重大，可否容我考虑一段时间，如果你现在就要逼我的话，我的实力你也是知道的，你要的是钱，不是跟我作对。”</span>
<span style="">这句话说到司空搏心里去了，他只求财，根本不想在云城市跟实力雄厚的韩氏家族作对，这样对他没有一丝好处。话已到此，他今天的目的也已经达到，聪明的司空搏知道事情要有个度，他拿出手机：“那劳烦兄弟告诉我你的手机号，两天之后我会跟你联系。”</span>
<span style="">韩戈说出一串号码，司空搏当场打过去，韩戈随身携带的手机响起，司空搏满意地点点头：“这样就好，兄弟，既然这次咱们能聚到一起，如果有什么意外，下次咱们还会用这种方式相见的，你是聪明人懂得我的意思，祝咱们以后合作愉快！”说完，司空搏起身要走。</span>
<span style="">“还是我带你们出去吧，这么高的悬崖，下去要比上来难得多。”韩戈大度地说道</span><span style="">。</span>
<span style="">“你果然仗义，不过你认为我们这一身装扮能安全走过摄像头吗？”司空搏微微一笑，仰身跳出窗口，引得若叶一声惊呼。</span>
<span style="">“再见！”水鬼色迷迷地冲若叶招招手，跟随着司空搏一跃而下。悬崖间，两张降落伞盛开在半空，飘摇而下。</span>
<span style="">韩戈对着若叶甜美一笑，悄悄擦了擦掌心里的汗水。</span>
<span style="">海边，司空搏两人收起降落伞后拿出手机，把韩戈的电话号码记下。司空搏抡起手臂高高一扬，手机呈抛物线落入海中，“砰！”他张开双臂夸张地模仿爆炸声。</span>
<span style="">“哥，你觉得有戏吗？”水鬼也被他的情绪感染。</span>
<span style="">“兄弟记住，只要有演员，就会有戏！”司空搏大笑着拍拍水鬼，冲自己的停车处走去。</span>
