<span style="">“好，快去快回，小心点！”绍辉回答道。</span>
<span style="">赵正豪披着伪装网跑进了雨雾，瞬间消失了身影。</span>
<span style="">二十分钟后，正当大家等得焦急时，赵正豪忽然出现在他们面前，雨太大，能见度不到两米。“跟我走，前面有个土洞！”</span>
<span style="">雨水汇集成灾，顺着地势澎湃而下。一行人趟着深浅不一的水坑跟随着赵正豪，来到了一处快要塌方的土洞前。</span>
<span style="">这应该是某种野禽掏出来的，被这场豪雨浇得变了形，同时扩宽了洞口。</span>
<span style="">几个人站在雨中，看看这个只能跪着进去的洞口，又看看赵正豪。</span>
<span style="">“爱进不进，我就这本事了！”赵正豪喊道。</span>
<span style="">左明摇摇头，把枪收好，拔出匕首率先钻了进去，随后里面传来几声扑腾：“进来吧，还算宽敞！”</span>
<span style="">众人依次进入，发觉这里是个天然洞穴，一棵参天大树的硕根将周围土壤割得支离破碎，然后被野猪稍作加工后，变成了野猪窝。</span>
<span style="">之所以这么肯定这里是野猪窝，因为它就在洞里。一头不大的野猪瞪着狭细的双眼正虎视着这帮不速之客，身下，是一窝刚刚长出毛的小崽子。再看看左明捂着头的姿势，可以判定一人一猪刚进行完一场争斗。</span>
<span style="">众人一致的意见是只是借宿，不进行人室犯罪活动，尤其是“上帝”表示，自己以往杀生过多，如今心已向善，天下众生皆平等，呜呼哀哉。</span>
<span style="">绍辉没理他这半佛半儒的话，发</span><span style="">动众人动手简单修葺了一番，以免半夜塌陷被活埋在树下还不知道。</span>
<span style="">完毕后，众人脱下湿透的衣服，合眼休息。洞内，只有野猪偶尔的喘气声，还有洞顶一处渗水滴落的“嗒嗒”声，除此，一片死寂。</span>
<span style="">不知过了多久，绍辉突然睁开眼，抬头侧耳细听。黑暗中，有人拍拍他的腿，轻声问道：“你也听见了？”</span>
<span style="">是赵正豪。</span>
<span style="">“有马队从上面经过。”绍辉小声说道，“大半夜下着雨，谁会赶路？”</span>
<span style="">“雨早停了，这帮人应该就是马仔，运毒的。”赵正豪又听了听，“这是个好事，等明天咱们顺着脚印走，肯定有收获的。”</span>
<span style="">“现在吧，这深山里天气不正常，如果再下一场雨的话，我怕什么痕迹都找不到了。”绍辉想了想，提议道。</span>
<span style="">“也好，他们的方向应该是山内，就算不是山佛的人，最起码会找到一条进山的路，我去把他们都叫醒。”赵正豪说完，探身拍拍左明和“上帝”，“嘘！出发！”</span>
<span style="">俩人乍醒之时便反应过来，压低声音：“有情况？”</span>
<span style="">“有马队经过，跟着他们走。”赵正豪在黑暗中指指上方说道。</span>
<span style="">几人略一收拾，钻出了野猪洞。</span>
<span style="">野猪在里面哼哼了几声，不知是送客还是炫耀自己的胜利。</span>
<span style="">洞外，密不透风的顶天枝叶连接在一起，不见天日，把本来就是黑夜的山林压得更加沉寂。四周，处处有积雨从树叶滑落，滴在水洼中溅起一</span><span style="">声声空灵的声响。</span>
<span style="">绍辉四人小心翼翼地踩着松软的地面，尽最大可能不发出任何声响，朝着马队方向慢慢摸去。很久之后，他们突然意识到：马队消失了，或者说，他们跟丢了！</span>
<span style="">四人立刻停下脚步，将自己伪装成四堆山草观察情况。这时的他们心里异常紧张，因为在这充满神秘危险的深山中，此刻，还有一群武装贩毒人员正在附近，稍有不慎，随时可能发生枪战。目前，这是他们最不想见到，也不愿遭遇的情景。</span>
<span style="">深夜的山中，万籁俱寂。</span>
<span style="">绍辉举目观察，除了黑暗，还是黑暗。他感觉这样下去不是办法，除非他们愿意在这水洼中趴上几小时，他决定动身去四周查看一番。于是，绍辉静悄悄地站起身，趁着树叶间洒下的斑点星光，一寸一寸，向身旁摸去，虽危险，总比拉着所有队友趴在泥泞中空耗体力要好一些。</span>
<span style="">来到一片山林中时，他忽然停下脚步，多年命悬一线的生存经验让他感应到：自己的身边有东西！准确些讲，身边还有一个人！</span>
<span style="">绍辉沉着地拔出匕首，万不得已之时，他不会选择开枪。</span>
<span style="">他的手渐渐握住刀柄时，一个人影同时也发现了他。对方反应奇速，第一时间扑了过来，随着身影压来的，还有一丝闪着寒光的刀刃。绍辉是搏击好手，拉长身体伸直手臂，站在原地最大限度地送出手中匕首在黑影面前划下一道。对方</span><span style="">的刀刃还未碰触到绍辉，便中了刀，然后快速地遁入林中。</span>
<span style="">短短一秒钟，两人相互搏命一击后，只剩绍辉站在了原地。</span>
<span style="">什么东西动作这么快？在他的意识里，刚才那人不能算人，只能说是一个移动速度很快的“东西”。</span>
<span style="">绍辉下意识地拔出手枪，一刀一枪护住身体转着圈看着四周。毫无疑问，刚才的对手是个人，如果一个人能有如此兽性的身手，那他将比野兽还要凶残可怕。</span>
<span style="">此地不宜久留，绍辉拔腿跑去跟队友会合，想必对方也不愿在途中发生不必要的伤亡，己方主动撤出，说不定可以化解这场已经搭箭在弦的火并。</span>
<span style="">他不知道，这支运输队正是山佛的手下。负责运输的，也就是刚才与他交手的，正是从小在深山老林中跑大的半兽人，绍辉此次追缉任务的主角之一——白夜。</span>
<span style="">“砰砰！”</span>
<span style="">绍辉跑动的声音引发了黑暗深处的枪声，烧红的弹头流星般划破老林的黑暗，瞬间没入土地或穿透树干不知去向。</span>
<span style="">同时，绍辉的队友也开了火，子弹斜着打人那几处枪火处，掀开了这场遭遇战的序幕。</span>
<span style="">“撤，别打无谓的战斗！”绍辉疾步来到赵正豪身边，短促说道。</span>
<span style="">“撤！”赵正豪喊了一句，四人持枪弓着身子，边打边依靠地形树木撤出了这片战场。</span>
<span style="">四人在伸手不见五指的森林中跑出很远，此时，东边天际开始微微泛白。</span>
<span style="">“那是什么人？”“</span><span style="">上帝”听完绍辉的描述后，被唬得一愣一愣的。</span>
<span style="">“难道是他？”绍辉突然想了起来，“你们想想，这人的本事跟资料里的白夜是不是很相似？”</span>
<span style="">众人一经提示，发觉此人还真像那个传说中的人猿。</span>
<span style="">“追！”左明提起枪便要原路返回。</span>
<span style="">“等等，太莽撞了。”赵正豪思忖着，“能在状况这么复杂的夜里发觉我们的跟踪，并且设伏围歼，还敢一人靠近进行无声暗杀，这个白夜是个对手，你觉得呢？”他问绍辉。</span>
<span style="">绍辉点点头：“还有种可能，我们已经暴露了目标。”</span>
<span style="">“那咋个治？”“上帝”嘴里又吐出一句自创方言。</span>
<span style="">“打呗。”赵正豪嘴里轻轻吐出一句话，“在这深山里，多点敌人和少点敌人，区别不大。”</span>
<span style="">“好了，咱们找个高处看看方向吧。”绍辉站起身，“早一天击毙目标，早一天回家休息，走吧。”</span>
<span style="">几个人爬上就近的山顶时，天已放亮。</span>
<span style="">下了一夜的雨，整片原始群山焕发出崭新的生机，苍茫茫，青翠欲滴，不知名的鸟虫正此起彼伏地叫着，点缀了这片充满野性的大好河山。</span>
<span style="">“你说，要是没有战争，该有多好！”左明大发感慨。</span>
<span style="">“闭嘴，过来看一下地图！”绍辉毫不客气地将他的诗兴扼杀掉。</span>
<span style="">赵正豪依据朝阳与地图，辨别好方向后重新制定了路线，继续出发。</span>
<span style="">深山老林的环境是可以杀人的。滂沱大雨过后又被太阳晒过，整</span><span style="">片森林如同桑拿室一般灼气逼人，尤其地表，虽有树木遮挡，但温度只高不低，四人好似用劣质汽油自焚未遂一般，浑身冒着白色蒸汽再次一步步潜进更深的山中。</span>
<span style="">前面，是一条宽阔的山河，有居民在河中汲水洗菜，另有孩童在其间戏水。</span>
<span style="">不远处，一片寨子静静矗立在夕阳的余晖中，安静祥和。</span>
<span style="">此时已经是黄昏。</span>
<span style="">四人静悄悄地趴在山坡望着眼前景象，却体验不到其中任何的安逸。</span>
<span style="">依据资料，这里已经到达了“怒海”队员们发生火并的地带，也就是说，山佛的老窝，正是在这附近。</span>
<span style="">“要不要下去打听一下？”绍辉轻轻问道。</span>
<span style="">赵正豪放下狙击镜，摇摇头：“不行，这些居民的背景咱们不了解，说不准他们是山佛的人，或者跟他有关系，咱们贸然下去，恐怕很难再上来。”</span>
<span style="">气氛沉默下来。此时，或许就是这么不经意的一个错步，自己就会变成一具尸体，受尽风吹雨淋后，腐烂成淤泥。</span>
<span style="">“等等吧，天黑后再下去摸一下，或许有些收获。”赵正豪端起狙击枪，重新观察着。</span>
<span style="">四人就这么警戒着周围，看着眼下，耐心等待着日薄西山。</span>
<span style="">天，渐渐暗下来。光着屁股的孩童跟随在大人身后，一摇一摆回了家。</span>
<span style="">河边恢复了平静。</span>
<span style="">“你们三个下去，注意安全，不行立刻撤回，我在这里掩护你们。”赵正豪看看仅存的夕阳一角，下了命令。</span>
<span style="">三人</span><span style="">披着伪装网悄悄爬起，持枪警戒着朝山下走去。</span>
<span style="">赵正豪的枪口在山下与对面山体上游荡着，寻找着一丝一毫可能致命的蛛丝马迹。</span>
<span style="">然而，危险就在身边。</span>
<span style="">白夜是何许人也！</span>
<span style="">从小在深山靠着猎杀野兽为生的他，想在这里悄无声息地埋葬几个鲜活的生命，实在太容易了。</span>
<span style="">那夜，他感到有人跟踪后，便设下埋伏准备围歼对方，不料对方并非泛泛之辈，只是苦于身处黑暗丛林。否则，白夜很有可能遭遇了从小到大以来在山中的第一次耻辱——能在山中跟他打个平手甚至高他一筹的人，来历和目的，绝不简单。</span>
<span style="">于是，他向基地报告后，选择了跟踪，准备在有人落单时，逐个将他们留在这片没有名字的山林中。</span>
<span style="">顺着绍辉三人的轨迹往上面的山坡看，远处的白夜露出了一丝笑意。</span>
<span style="">山坡上，赵正豪正聚精会神地观察着下面的情况，突然，他听到身后传来一丝异响。这种声音，只有压强和受力面积达到一定程度后踩到草丛时才能发出。</span>
<span style="">在山中，能发出这种声音的只有大型猛兽，或者是人。</span>
<span style="">赵正豪不动声色地将手在扳机处移开，慢慢滑向腰间匕首处。这时，背后的异响换成了一阵急促风声扑来。赵正豪一凛，急忙翻身离开原位置，同一时间，一个黑影扑在了他趴的位置，一柄乌黑的匕首，正不偏不倚地扎在刚才他所在的土地上。</span>
<span style="">白夜偷袭未</span><span style="">成，趴在地上侧过脸冲赵正豪阴阴地笑着。</span>
<span style="">山下，绍辉三人已经下了河准备泅渡进寨。黄昏的山河表面泛着一层暖暖的金黄色，看似柔顺平缓的河面内去暗流湍急，好像有无数条水蟒缠着双腿，拼了命把他们向未知的方位带去。幸好三人水性都不差，即使最差的“上帝”，也能勉强保护着枪支进行武装泅渡。</span>
<span style="">“噗通……噗噗……”正当绍辉刚看完“上帝”的情况转过头时，“上帝”就开始不争气地溺水了，他的脑袋在宽阔的河面上沉浮，双手开始拼命拍打着水花。</span>
<span style="">“快去帮他！”绍辉对左明喊道。</span>
<span style="">“你娘……有人在……下面拉……我！”“上帝”探出头，断断续续骂了这一句。</span>
<span style="">绍辉大惊，急忙潜下水去查看。水下的暗流令人很难睁开眼，朦胧间，他看到一个黑影正拽着“上帝”的双腿，玩儿命地将他往深水里拖。</span>
<span style="">绍辉摸出手枪扣下扳机，一声闷响，枪机一张一合挤出一串水泡后，一粒弹头撕裂着河水，留下一圈圈螺旋形弹道冲人影飞速钻了过去。人影见状，立刻松开手，身形下沉避开子弹，消失在黑青色的河流深处。</span>
<span style="">“水鬼，回撤！”绍辉冒出水面，嘶声喊了一句。</span>
<span style="">此时，河对岸出现了一些持枪人员，随着一阵密集的枪声，水面如同下了一阵急雨般，簌簌溅起水花。</span>
<span style="">但是，作为队友火力掩护的那支狙击枪，却在这关键</span><span style="">时刻没了动静。</span>
<span style="">赵正豪的身上此刻已有几道刀伤，而白夜的胳膊上也有血滴渗出。</span>
<span style="">当年司空搏和水鬼一直劝白夜学习一下枪械使用，而他从小只认猎刀，猎枪都不曾使用过。如果自己会使用枪支，如果刚才一枪毙了这个狙击手，现在就不会这么费力了。</span>
<span style="">赵正豪寻找着机会去摸腿间的手枪，而白夜盯着他，防止他拿到枪。</span>
<span style="">血，一滴滴滑落，俩人就这么站在丛林中，对峙着。</span>
<span style="">“小子，好本事！哪儿学的？”白夜问道。</span>
<span style="">“中国解放军特种大队。你也不错啊！”赵正豪针锋相对。</span>
<span style="">他的话刚落，白夜趁机发难，倒转猎刀甩了出去，猎刀拉出一条直线飞速射向赵正豪的头部。更为恐怖的是，白夜手中又多出一柄刀，身形紧随着飞刀之后，几乎是在同一时间来到了赵正豪的面前。</span>
<span style="">或许他知道赵正豪穿着防弹衣，两柄刀，上迎面骨下挑裆部，瞬间发动了致命攻击。</span>
<span style="">赵正豪反应过来，但已经躲不过了，他只有咬牙侧过身，飞刀穿透脸面扎进了他的牙床，另一把猎刀，轻轻松松划开了他的大腿外侧。</span>
<span style="">赵正豪踉跄倒地时终于有机会拔出手枪，白夜见状，野兔般消失在了茫茫苍林中。这时，忽然又有人影在林中出现，他把枪口移向此处时发现，是绍辉他们。</span>
<span style="">“老赵，你怎么了！”绍辉急忙上前扶起赵正豪。</span>
<span style="">“撤，中埋伏了！”赵正豪捂着面部喊</span><span style="">道。</span>
<span style="">四人又是一路狂奔，赵正豪的腿部虽然受伤，但这点疼痛丝毫不妨碍他跑步的速度。几人一直跑出很远已经听不到枪声之后，才选了一个不易察觉的角落，开始为他包扎伤口，稍作休息。</span>
<span style="">“他奶奶个大腿的，老衲差点被那孙子淹死了！”“上帝”的中文风格很别致，脑袋里有啥词，嘴里就往外蹦啥话。</span>
<span style="">这次没人笑话他。</span>
<span style="">只是刚刚一照面就差点死了两名队员，每个人对于前景都不是很乐观。</span>
<span style="">夜幕又降临在这片神秘的山区之上，树木变得狰狞起来。远处的山仿佛是一头已经张开血盆大口的猛兽，虎视眈眈地看着这四名不速之客。</span>
